A simple and secure Password Generator Web Application that creates strong random passwords to help users protect their online accounts.

---

📌 Project Overview

Passwords are important for protecting personal data online. Weak passwords can easily be hacked. This project generates strong and random passwords using JavaScript to improve security.

The application is lightweight, easy to use, and runs directly in the browser.

---

✨ Features

- 🔑 Generate strong random passwords
- ⚡ Fast and simple interface
- 💻 Works directly in the browser
- 🔒 Helps improve online security
- 🎯 Beginner-friendly project

---

🛠️ Technologies Used

- HTML – Structure of the webpage
- CSS – Styling and layout
- JavaScript – Password generation logic

🚀 How to Run the Project

1. Clone the repository

git clone https://github.com/manikshitagolla-byte/password-generator.git

2. Open the folder

cd password-generator

3. Run the project

Open "index.html" in your browser.

---

💻 Usage

1. Open the application.
2. Click the Generate Password button.
3. A secure password will be generated instantly.
4. Copy the password and use it for your accounts.
